OpenClaw 3.28 Is Here: 3 Major Updates You Need to Know

OpenClaw just got a massive upgrade! From one-click skill installs to smarter AI web search, here is everything you need to know about version 3.28.

📅 March 26, 2026 · ⏱ 2 min read

🧠 Get 1:1 help from a Software Engineer to automate your workflow → https://www.skool.com/ai-academy-with-robby-6849/about

Hi everyone! I’m Robby. As an AI engineer, I spend a lot of time testing new tools. This week, the team behind OpenClaw dropped some huge updates (versions 3.24 to 3.28), and honestly, it’s a game changer for building AI agents.

If you want to keep your workflow fast and smart, here is the breakdown of what is new.

1. ClawHub: Skills at Your Fingertips

Before, finding and adding new "skills" to your AI was a bit of a chore. Now, you can browse and install skills directly inside OpenClaw using ClawHub. It’s like an app store for your AI—just click, install, and go.

2. The /btw Command

Have you ever had your AI get confused because it had too much "stuff" in its memory? That’s called context pollution. The new /btw command helps you manage context better. It’s a clean way to tell your AI extra details without cluttering the main conversation.

3. Smarter Web Search

You can now connect your AI to the live internet! By using plugins like FireCrawl, Exa, or Tavily, your agents can look up real-time data.

How to set it up:

  • Open your Control UI.
  • Find the API settings for your chosen search provider.
  • Plug in your API key.
  • Enable "Web Fetch" in your agent settings.

4. Thinking Modes and GPT-5.4

OpenClaw updated its default model to GPT-5.4, which is smarter and faster. Plus, there is a new setting for "Thinking Modes." You can now switch how your agent thinks depending on the task. Use a faster mode for simple chats and a deeper "thinking" mode for complex coding projects.

5. One-Click Skill Installs

Managing tools used to be hard, but now the Control UI makes it easy. You don’t need to write complicated code to get a new tool running. Just look for the one-click install button in the UI, and your agent will be ready to work in seconds.

Final Thoughts

AI is moving super fast right now. The best way to keep up is to experiment with these new features. Try installing a new skill from ClawHub or testing out the web search feature today.

Happy coding!
